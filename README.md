# 🏋️ GymManagement – Desktop Gym Management System

GymManagement is a desktop application built to help gyms efficiently manage members, subscriptions, and daily administrative operations through a simple and modern graphical interface.

---

## 📌 Overview

This project is designed to simulate a real-world gym management system where administrators can handle member records, track subscriptions, and organize gym activities in an efficient and structured way. The application focuses on usability, clean design, and practical data management.

---

## 🛠️ Tech Stack

- Python  
- CustomTkinter  
- Tkinter  
- SQLite

---

## ✨ Features

### 👤 Member Management
- Add new members
- Update member information
- Delete member records
- View complete member list

---

### 📅 Subscription System
- Create and manage subscriptions
- Track active and expired memberships
- Assign plans to members

---

### 📊 Admin Dashboard
- Overview of gym statistics
- Quick access to main features
- Organized and user-friendly interface

---

### 🎨 UI / UX
- Modern GUI using CustomTkinter
- Clean and simple layout
- Easy navigation between sections
